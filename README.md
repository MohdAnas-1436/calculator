## 🎀 Welcome to the Calculator!

Welcome to **Mohd Anas's** CLI Calculator! This isn't just any ordinary calculator—it's a fun, interactive, and cleanly coded way to perform your daily arithmetic right from the command line. 

---

## ✨ Features

* **➕ Addition:** Bring numbers together!
* **➖ Subtraction:** Find the difference!
* **✖️ Multiplication:** Multiply the fun!
* **➗ Division:** Safe splitting with built-in zero-division error handling.
* **🎮 Interactive Menu:** Easy-to-use menu operation that loops continuously.
* **🛡️ Smart Validation:** Catches invalid inputs gracefully so the app never crashes on you.

---

## 🚀 How to Run

Ready to crunch some numbers? Make sure you have Python 3.x installed, pop open your terminal, and run:

    python calculator.py

*Just follow the prompts to select your operation and input your numbers. The math magic happens instantly! 🪄*

## 🧠 Code Highlights

* **Modular Design:** Clean, separate functions for `add`, `subtract`, `multiply`, and `divide`.
* **Foolproof Error Catching:** Built-in safeguards against division by zero and wonky text inputs.
* **Continuous Loop:** The program keeps running until you explicitly tell it to exit—perfect for long study sessions!
